# Breast Cancer Management Review

Welcome to the repository for our narrative review: **Contemporary Breast Cancer Management in the Era of Precision Oncology: Therapeutic Innovations, Barriers to Care, and Future Directions**.

## Repository Structure
- `Manuscript_Draft_Part[1-3].md`: The core text of the review.
- `Manuscript_Tables.md`: The 5 structured clinical tables.
- `References.md`: The final Vancouver-formatted bibliography (47 citations).
- `Master_Reference_Database.md`: The verified database containing PMIDs/DOIs for all citations.
- `Figure1_Timeline.png` through `Figure5_Roadmap.png`: Custom-built Matplotlib figures mimicking BioRender quality.
- `Handover_Context.md`: **IMPORTANT!** Read this file first. It contains the complete history, decisions, and exact next steps required to finish the paper.

## Quick Start
1. Read `Handover_Context.md` to understand the current phase of the project.
2. Begin compiling the markdown files into your preferred word processor.
3. The citations are fully audited and sequenced. Please do not reorder sections without re-running a Vancouver sequence script!
